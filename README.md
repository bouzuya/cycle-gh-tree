# cycle-gh-tree

A GitHub issue browser in [Cycle.js](https://github.com/cyclejs/cycle-core).

## Installation

```
$ git clone https://github.com/bouzuya/cycle-gh-tree
$ cd cycle-gh-tree
$ npm install
```

## How to use

```
$ # run cycle-gh-tree & open in browser
$ npm run watch
```
